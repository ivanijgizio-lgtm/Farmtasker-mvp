# FarmTasker - Агрегатор фермерских заданий и свежих продуктов

## Описание
FarmTasker — веб-приложение-агрегатор для поиска и заказа свежих агро-продуктов с российских ферм. Основная функция: поиск ферм по продуктам/регионам, задания на выращивание и доставку. Вдохновлено [funnyferma.ru](https://funnyferma.ru/) — фокус на свежесть, регионы, доставку пн/ср/пт.

**Тренды 2026 РФ:** Импортозамещение агро, экология, онлайн-заказы с ферм.[web:20]

## Функции (Front-End MVP)
- **Неоморфный дизайн** (neumorphism): мягкие тени, градиенты, blur.
- **Адаптивный** (mobile-first): работает на всех устройствах.
- **Поиск:** Input + фильтры (органика, доставка, задания).
- **Карточки ферм:** Примеры с ценами, регионами (как Волжская Ферма).
- **Мотивация:** Слоганы "Свежие продукты с ферм под заказ!".
- **Иконки:** Font Awesome (семена, трактор, корова).
- **Интерактив:** Hover-эффекты, активные фильтры, поиск по Enter.

## Технологии
- **HTML5/CSS3/JS** (чистый, без фреймворков).
- **Font Awesome 6.5** (иконки).
- **Google Fonts (Inter)** — современный шрифт.
- **Градиенты/Backdrop-filter** — для neumorphism.
